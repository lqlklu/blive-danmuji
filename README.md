# blive danmuji

## 快速开始

### 1. 安装

解压 `blive-danmuji-win.zip`或`blive-danmuji-linux.tar.gz`

运行 `setup.ps1`(Windows) 或者 `setup.sh`(Linux)。

### 2. 解压模型

解压到当前目录下，有两个文件 `G_latest.pth` 和 `finetune_speaker.json`。

### 3. 运行

运行 `main.ps1`(Windows)或 `main.sh`(Linux)，脚本中的参数视情况修改。
